# PvZ2中国版创意庭院 JSON文档
### 贡献说明
- 请在/docs文件夹中写文档,具体请参照[docsify文档](https://docsify.js.org/#/zh-cn/quickstart)

### 本地预览
- 请安装docsify-cli
```
npm i docsify-cli -g
```
- 然后执行下面命令即可本地预览
```
docsify serve docs
```

### commit 书写说明
- 按照<type>: <subject>的格式书写commit,请养成及时提交更改的习惯
- 下面是可能用到的type
```
feat: 新增
fix: 修改
del: 删除
init: 初始化
```
